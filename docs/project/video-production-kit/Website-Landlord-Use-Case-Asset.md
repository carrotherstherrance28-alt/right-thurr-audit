## 9:16 Script (35–50s, Draft)

Style rules:

- Speak in “can / model / example / depends on market” language.
- Never promise “page one”, lead volume, revenue, profit, or retention.
- Use placeholder niches/cities (ex: “Junk Removal Austin”) and fake data only.

Shot timing: ~4–6 seconds each.

1) Hook — Own the lead path

- On-screen: “Own the lead path.”
- VO: “Here’s a simple model: you can own the lead path without owning the service business.”

2) Pick niche + city

- On-screen: “Niche + city first.”
- VO: “Start narrow: one niche in one city.”

3) Build the site for conversion

- On-screen: “Build for calls + forms.”
- VO: “Build a site that’s designed for calls and form inquiries.”

4) Earn visibility (carefully)

- On-screen: “Earn local visibility.”
- VO: “Visibility can be earned over time—no guarantees, it depends on the market.”

5) Track the calls/forms

- On-screen: “Track every inquiry.”
- VO: “Tracking is the difference between a hobby and an asset.”

6) Route to a local operator

- On-screen: “Route leads to an operator.”
- VO: “Then route the inquiries to a local operator who can actually fulfill the work.”

7) Flat monthly rent (positioning)

- On-screen: “Flat monthly is common.”
- VO: “Many deals are a flat monthly fee. The right number depends on lead quality and demand.”

8) Trial / proof

- On-screen: “Let the calls prove value.”
- VO: “If you’re doing it right, the calls prove value before anything long-term.”

9) Right Thurr angle (template)

- On-screen: “Copy • Curate • Rebrand”
- VO: “Right Thurr is the template. Copy it, adapt it, brand it—then make the system actually work.”

10) CTA (Thurr Solutions)

- On-screen: “Want this mapped for your niche? Start with the audit.”
- VO: “If you want this mapped for your niche, start with the Lead Flow Audit.”

## Runway / Generation Prompts (Draft)

General settings:

- Format: 9:16
- Output: clean motion backgrounds; add final text/UI overlays in Remotion for accuracy.
- Avoid: real Google UI, real phone numbers, real client identifiers, “guaranteed” language.

Prompt style: “clean editorial motion graphic” + “paper + ink + clay orange” + “bold black border”.

Per-shot prompts:

1) Hook (poster-like)
- Prompt: “minimal editorial motion graphic, paper background, bold black border frame, clay-orange accent, large headline area (blank), subtle push-in camera, professional high-trust look, no logos, no photos”

2) Niche + city selection
- Prompt: “clean tile grid of niches and cities, paper texture, ink typography placeholders, one tile highlights in clay orange, simple 2D motion, no real map labels”

3) Website sections
- Prompt: “abstract website layout blocks, hero + call button placeholder, form placeholder, review cards, gentle downward pan, paper background, ink outlines, clay-orange CTA block”

4) Visibility concept (abstract)
- Prompt: “abstract search visibility concept, generic result cards (not Google), a simple meter rising from low to medium, paper background, ink outlines, restrained motion”

5) Tracking dashboard (generic)
- Prompt: “generic call tracking dashboard UI (fictional), cards sliding in labeled ‘New call’ and ‘New form’, no real numbers, paper/ink palette with clay-orange tag”

6) Routing diagram
- Prompt: “simple routing diagram: website icon → tracking icon → operator icon, one lead card animates across arrows, bold border frame, clean 2D vector look”

7) Economics card (no promises)
- Prompt: “economics card layout with labels ‘flat monthly’ and ‘depends on market’, numbers area blank, paper background, ink typography, clay-orange underline sweep”

8) Trial / toggle
- Prompt: “simple toggle switch animation ‘forwarding on’ (generic), trial card, clean editorial 2D design, paper texture”

9) Template library + system map
- Prompt: “two-panel composition: left is a ‘use case’ page placeholder, right is a 4-step system map, clay-orange badge ‘copy’, clean motion graphic push-in”

10) CTA brand card
- Prompt: “final brand card placeholder, bold border, clay-orange accent bar, clean editorial motion, space for CTA text, no logos”

## Compliance Reminders (Non-Negotiable)

- No promises: rankings, leads, revenue, passive income, retention, profit.
- No real client names, phone numbers, inboxes, screenshots, or ad accounts.
- Use “example / template / depends on market” language throughout.
- If quoting any economics publicly, require explicit approval + disclaimers.
