# Quality Checklist (AI Video Clips)

Use this before marking a clip “usable” and before any client review/publish.

## Visual Fidelity

- Subject stays stable (no morphing faces/hands/logos).
- Brand assets remain recognizable.
- Motion is intentional (not chaotic).
- Lighting/textures feel grounded for the brand.

## Brand Consistency

- Tone matches brand (not default “AI luxury”).
- Visual anchors are obvious and consistent.
- Avoids off-brand styles listed in intake.

## Text / Readability

- Any overlay text is readable on mobile.
- If generation made text messy: remove it and add text in Remotion/manual edit.

## Platform Fit

- Hook is clear in the first second.
- Clip pacing matches target (website vs social).
- Export format(s) match deliverables (16:9 / 9:16 / 1:1).

## Professionalism / Compliance

- No sensitive client data appears in dashboards/screenshots.
- No unapproved claims (especially medical/financial).
- No voice cloning or model training without explicit approval/rights.

## Pass/Fail Call

- Pass:
  - Keep and assemble.
- Fail:
  - Apply retry notes (reduce motion, strengthen anchor, regenerate).
