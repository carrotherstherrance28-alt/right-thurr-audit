# Pilot Shot Plan — Thurr Solutions Lead System Intro (6–10 shots)

Use this as the first-pass “translator” document for generating clips (Runway) and assembling the website intro (Remotion).

Scope: buyer-facing explainer of the Lead Flow Audit + build + managed retainer. No client names, no private data, no “AI tool” callouts.

## 1) Creative Direction (Translator)

### Brand Tone

Pick 2–3:

- high-trust/professional
- practical
- premium

Tone notes:

- What it should feel like in 1 second: calm control — “this is a real operations system.”
- What it should *not* feel like: sci-fi AI, course-seller hype, stock-luxury fluff.

### Visual Anchors (Non-Negotiables)

- Anchor 1: “Lead path” system map (simple pipeline: inquiry → intake → owner alert → follow-up → next step).
- Anchor 2: Audit dashboard UI (warm paper + black panels + orange status markers; checklist highlights biggest leak).
- Anchor 3: Clear CTA frame (“Start with the Lead System Audit.”) with Thurr Solutions lockup.

### Pacing Pattern

- calm explainer → steps → CTA

### Camera / Motion Logic

Rules:

- One camera move per shot.
- One clear subject per shot.
- Prefer grounded motion (push-in, slow pan, gentle orbit).
- Text overlays must stay large and readable on mobile.

Allowed camera moves:

- locked camera
- slow push / dolly in
- gentle pan (left/right)
- slow pull back
- close-up insert push

### Quality Boundaries (Must Avoid)

- morphing logos, hands, faces
- unreadable overlay text
- neon AI / robot visuals
- overly glossy HDR / plastic smoothing
- cluttered UI montage with no single focal point

## 2) Shot Plan

Target: 45–60 seconds total. Keep each shot ~4–7 seconds and let the voiceover carry nuance.

### Shot 1 — “The clock starts”

- Goal (what it proves / what it should make them feel): Leads show up everywhere; speed + clarity matters.
- Input asset (screenshot/photo/logo/graphic): Simple “incoming lead sources” tile wall (icons: website, form, DM, call, referral).
- Visual anchor (what must stay stable): A single “New inquiry” card centered.
- Camera motion (one move): slow push in.
- Scene motion (what moves in the scene, if anything): One subtle notification pulse on the inquiry card.
- Overlay text (short, mobile readable): “A new inquiry comes in.”
- Tool recommendation: Runway.
- Retry notes (what to adjust if it drifts): Reduce background motion; keep icons crisp; keep card edges stable.

### Shot 2 — “Most leaks are in the path”

- Goal (what it proves / what it should make them feel): The problem is usually the response path, not “more leads.”
- Input asset (screenshot/photo/logo/graphic): Minimal system map (inquiry → intake → owner alert → follow-up → booked/next step).
- Visual anchor (what must stay stable): The pipeline nodes + arrows.
- Camera motion (one move): gentle pan left-to-right following the flow.
- Scene motion (what moves in the scene, if anything): A single highlighted dot travels through the pipeline.
- Overlay text (short, mobile readable): “If the path is unclear, warm leads go cold.”
- Tool recommendation: Remotion (preferred) or Runway.
- Retry notes (what to adjust if it drifts): If Runway morphs arrows/text, rebuild in Remotion with vector assets.

### Shot 3 — “Lead Flow Audit”

- Goal (what it proves / what it should make them feel): Thurr Solutions starts with an audit — focused, practical.
- Input asset (screenshot/photo/logo/graphic): Audit dashboard UI (warm paper + black panels + orange status markers).
- Visual anchor (what must stay stable): A “Biggest leak” row and a simple checklist.
- Camera motion (one move): slow push in to the highlighted “biggest leak” line.
- Scene motion (what moves in the scene, if anything): One checkbox ticks; one orange marker locks in.
- Overlay text (short, mobile readable): “Start with a Lead Flow Audit.”
- Tool recommendation: Runway.
- Retry notes (what to adjust if it drifts): Remove any tiny text; keep it as bold labels only (2–4 items max).

### Shot 4 — “What we review”

- Goal (what it proves / what it should make them feel): The audit covers the full chain, not just the homepage.
- Input asset (screenshot/photo/logo/graphic): 4-card grid: website clarity, booking path, intake form, follow-up timing/owner visibility.
- Visual anchor (what must stay stable): The 4-card grid layout.
- Camera motion (one move): slow pull back to reveal the full grid.
- Scene motion (what moves in the scene, if anything): Each card gets a quick orange “scan” sweep (subtle).
- Overlay text (short, mobile readable): “Website. Booking. Intake. Follow-up.”
- Tool recommendation: Remotion (preferred) or Runway.
- Retry notes (what to adjust if it drifts): Avoid micro text; keep each card title 1–3 words.

### Shot 5 — “If the website is the leak…”

- Goal (what it proves / what it should make them feel): There’s a concrete build path when the site is the issue.
- Input asset (screenshot/photo/logo/graphic): “Growth Website + Intake” title card + a clean form screenshot.
- Visual anchor (what must stay stable): The form layout (name/phone/email + short intake question).
- Camera motion (one move): close-up insert push into the form CTA button.
- Scene motion (what moves in the scene, if anything): Cursor tap on CTA; button highlights once.
- Overlay text (short, mobile readable): “We build the intake path.”
- Tool recommendation: Runway.
- Retry notes (what to adjust if it drifts): Replace cursor with a simple highlight pulse if pointer morphs.

### Shot 6 — “If follow-up is the leak…”

- Goal (what it proves / what it should make them feel): Follow-up becomes automatic and trackable.
- Input asset (screenshot/photo/logo/graphic): Simple “owner alert” + “follow-up sequence” UI (no real phone numbers/emails).
- Visual anchor (what must stay stable): A single alert card + 2–3 follow-up steps.
- Camera motion (one move): gentle pan downward (alert → steps).
- Scene motion (what moves in the scene, if anything): One “sent” checkmark appears on step 1; step 2 queues.
- Overlay text (short, mobile readable): “We build the follow-up automation.”
- Tool recommendation: Runway or Remotion.
- Retry notes (what to adjust if it drifts): Use placeholder data only; keep copy generic (no client industry specifics here).

### Shot 7 — “Managed monthly”

- Goal (what it proves / what it should make them feel): Ongoing support keeps it working as revenue grows.
- Input asset (screenshot/photo/logo/graphic): Calm dashboard view: workflow health, lead status, alerts, “next action.”
- Visual anchor (what must stay stable): The “Next action” panel.
- Camera motion (one move): slow push in to “Next action.”
- Scene motion (what moves in the scene, if anything): One alert resolves; one status turns orange → green (subtle).
- Overlay text (short, mobile readable): “Then we keep it working.”
- Tool recommendation: Runway.
- Retry notes (what to adjust if it drifts): Don’t show dense tables; keep it to 3–4 tiles max.

### Shot 8 — “The core line”

- Goal (what it proves / what it should make them feel): Simple offer ladder, easy to remember.
- Input asset (screenshot/photo/logo/graphic): Bold text card with 3 beats + minimal icon accents.
- Visual anchor (what must stay stable): The three-line statement (no fancy warping).
- Camera motion (one move): locked camera.
- Scene motion (what moves in the scene, if anything): Each line fades in, one-by-one.
- Overlay text (short, mobile readable):
  - “The audit tells us what to fix.”
  - “The build fixes it.”
  - “The retainer keeps it working.”
- Tool recommendation: Remotion (preferred for crisp type).
- Retry notes (what to adjust if it drifts): If Runway text warps, do this shot entirely in Remotion.

### Shot 9 — “CTA / brand lockup”

- Goal (what it proves / what it should make them feel): Clear next step, confident close.
- Input asset (screenshot/photo/logo/graphic): Thurr Solutions end card (logo lockup + CTA).
- Visual anchor (what must stay stable): Logo lockup + CTA.
- Camera motion (one move): slow push in.
- Scene motion (what moves in the scene, if anything): Subtle orange underline sweep under “Lead System Audit.”
- Overlay text (short, mobile readable): “Start with the Lead System Audit.”
- Tool recommendation: Remotion.
- Retry notes (what to adjust if it drifts): Keep the end card clean; avoid background clutter.

## 3) Assembly Notes (If Using Remotion)

- Intro/outro brand frames: Use Shot 1 as the opener; end on Shot 9 with the clean lockup.
- Safe text zones (mobile): Keep overlay text centered and within the middle 60% width; avoid bottom 20% (UI overlays on platforms).
- Caption rules (optional): If captions are added, do 1 line max, ~6–10 words, no punctuation-heavy blocks.
- Export variants: 16:9 (website) + 9:16 (Reels) by re-framing shots (don’t shrink text).
