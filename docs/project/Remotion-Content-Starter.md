# Remotion Content Starter

Remotion is installed for programmatic video content, short-form posts, and
prototype clips.

## Commands

Open Remotion Studio:

```bash
npm run remotion:studio
```

Render the first vertical intro:

```bash
npm run remotion:render:intro
```

Output:

```txt
out/thurr-intro.mp4
```

## First Composition

Composition id:

```txt
ThurrIntro
```

Format:

```txt
1080 x 1920
30 fps
6 seconds
```

Use this first for:

- Instagram Reels
- LinkedIn vertical clips
- Discord prototype previews
- quick branded intros before walkthrough videos

## Content Direction

Keep clips focused on one business-building idea:

- missed leads
- slow follow-up
- manual proposals
- owner visibility
- AI buildout plans
- systems versus ideas

The first family of clips should prove the same product promise as the website:

```txt
Ideas do not pay you. Systems do.
```
