# Runway Lead System Intro Prompts

Use these prompts to create the visual layer for the Thurr Solutions intro video. Keep the visual style professional and grounded. Avoid neon AI imagery, robots, sci-fi tunnels, fake holograms, or anything that makes the offer feel unserious.

Production workflow reference:

- Phase 1 kit overview: `docs/project/Branded-AI-Video-Production-Kit.md`
- Templates: `docs/project/video-production-kit/Runway-Prompt-Templates.md` and `docs/project/video-production-kit/Motion-Direction-Template.md`

## Recommended Output

Create a 45-60 second 16:9 website intro video. Export an additional 9:16 vertical version for social.

Voice requirement:

Use Thurr's approved voiceover, not a generic stock or AI narrator. If Runway asks to generate or clone a voice, pause and confirm authorization before uploading or creating the voice. Voice samples should stay private and should not be placed in the public website folder.

Final website file target:

`public/media/thurr-solutions-lead-system-intro.mp4`

Poster image target:

`public/media/thurr-solutions-lead-system-intro-poster.jpg`

## Visual Direction

Style:

Clean B2B systems walkthrough. Warm paper background, black interface lines, orange highlights, subtle grid, dashboard panels, forms, arrows, and owner alerts. It should feel like a polished operations explainer for local service businesses.

Do not use:

- Futuristic robots
- Neon cyberpunk
- Fake 3D avatars
- Overly cinematic stock footage
- Dark abstract AI clouds
- Course seller cash graphics

## Scene Sequence

### Scene 1: The Problem

Prompt:

Clean animated business workflow scene on warm paper background. A local business receives leads from website, Instagram, referral, and contact form. Some leads fade as response time gets slower. Use black interface lines, orange highlights, subtle grid texture, professional B2B style.

Overlay text:

Warm leads go cold when follow-up is slow.

### Scene 2: The Audit

Prompt:

Professional system audit dashboard showing website clarity, booking path, intake form, follow-up timing, and owner visibility. A checklist highlights the biggest revenue leak. Warm paper UI, black panels, orange status markers, clean modern operations design.

Overlay text:

Start with the Lead Flow Audit.

### Scene 3: Website + Intake

Prompt:

Service business website screen transforms into a clear offer page with a lead form, booking path, gallery/proof section, and SEO checklist. The form submits into a lead record. Clean interface animation, warm neutral palette, orange highlights.

Overlay text:

If the website leaks trust, fix the intake path.

### Scene 4: Automation Build

Prompt:

Lead record moves through automation steps: CRM or spreadsheet, owner alert, follow-up sequence, AI summary, and booked appointment. Show arrows connecting each step in a clean dashboard map. Professional workflow animation, no sci-fi.

Overlay text:

If follow-up leaks revenue, automate the path.

### Scene 5: Managed Support

Prompt:

Owner dashboard showing workflow health, lead status, latest follow-up, alerts, and monthly improvements. Calm professional system view with orange success markers and clear next action. Warm paper and black UI style.

Overlay text:

The retainer keeps the system working.

### Scene 6: Final Card

Prompt:

Clean final brand card for Thurr Solutions. Warm paper background, black text, orange accent, simple system line graphic. Premium but practical B2B identity.

Overlay text:

The audit tells us what to fix. The build fixes it. The retainer keeps it working.

CTA:

Request a Lead System Audit.

## Voiceover Script

Voice direction:

Use Thurr's voice. Delivery should be calm, direct, confident, and practical. It should sound like a systems architect explaining the business case, not a hype ad.

Most local businesses do not only need more leads. They need a cleaner path for the leads they already get.

A new inquiry comes in from your website, booking page, Instagram, referral, or contact form. Then the clock starts. If response is slow, the details are scattered, or follow-up depends on memory, warm leads go cold.

Thurr Solutions starts with a Lead Flow Audit. We review your website, booking path, intake form, follow-up, and owner visibility. Then we identify the biggest leak closest to revenue.

If the website is the issue, we build the Growth Website and Intake path. If follow-up is the issue, we build the automation. If the system becomes business-critical, we manage it monthly.

The audit tells us what to fix. The build fixes it. The retainer keeps it working.

Start with the Lead System Audit.
