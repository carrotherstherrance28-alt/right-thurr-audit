# Unbranded Diagnostic Report Template

This template is built for client-ready diagnostic reports that can be re-skinned for any business, niche, or campaign. Keep the structure. Replace the brand layer.

Use active language. Be plain about the problem, the upside, and the next move. The report should feel useful before it feels impressive.

### Re-Skin Checklist

Before sending, replace:

- Logo, cover mark, favicon, and footer identity
- Primary color, accent color, typography, and button style
- Report name and CTA language
- Sender identity, contact information, social links, and legal text
- Any implementation partner attribution that does not belong in a white-label handoff

### Expected Outcome

## Diagnostic Scoring

Use a 1-5 score unless the client needs a different scale.

| Score | Meaning |
|---|---|
| 1 | Missing or actively hurting conversion |
| 2 | Present, but weak or inconsistent |
| 3 | Usable, but not yet dependable |
| 4 | Strong, with a few fixable gaps |
